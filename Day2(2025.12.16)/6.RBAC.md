# Role-based Access Control (RBAC)

### web console창 여는 법

```code
oc whoami --show-console
```

이후 나오는 링크를 마우스 우클릭 후, openlink해서 열기
나오는 console창의  
1번 버튼: kubeadmin 권한, 2번 버튼: LDAP권한, 3번 버튼: htpasswd권한

### Role 할당시 get,list,watch 권한 차이

oc get pods: list 권한
oc get pod nginx : get 권한
oc get pods -o watch(pod가 바뀌는 상태를 실시간 모니터링) : watch 권한
